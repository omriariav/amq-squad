# amq-squad v2.16.0

Draft release notes for milestone 2.16.0.

## Highlights

- Fixed Claude worker preauthorization so launcher-owned `--allowedTools` does
  not suppress generated bootstrap prompts, including cross-cwd workers launched
  from a shared team home. Team `up --dry-run --json` now exposes additive
  `child_args`, `preauthorized_actions`, and `bootstrap` fields on each
  `team_plan` member for launch-preview auditing (#318, #319).
