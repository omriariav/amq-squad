# amq-squad 2.0.0 — plan & tracking

One breaking release bundling the lifecycle/session redesign (#50) **and** Mission Control
(read-only incident-console TUI). Maintainer decision: ship together in 2.0.0.

**Gated:** all work lands on `release/2.0.0`; install from the PR, validate, then merge.
Each slice is Codex DX-tested (fixtures for mutating verbs, live data for read-only views)
before it's considered green.

## Decisions locked

- **`stop` primary, `down` deprecated alias.** `stop` = SIGTERM + reap wake + presence-flip
  (process stops, on-disk state preserved → resumable). `--force` = SIGKILL escalation.
  This replaces the broken "gentle pause" (which would have half-killed live agents).
- **`resume` = re-orient** for fresh seats (`conversation:""`): re-runs bootstrap so the agent
  reads the brief + drains AMQ history. Records *with* a saved conversation still skip bootstrap
  (true reattach). True conversation-continuity (harvest the id at stop-time) is tracked as a
  fast-follow — `resume` help states plainly it re-orients, not continues, for fresh seats.
- **`up <name>` = NEW work:** requires a brief (`--seed-from`/`--brief`), with `--no-brief`
  escape hatch for CI/tmux automation; refuses an existing session (→ `resume` / `up --reset`).
  `--reset` is confirm-gated (it tears down live agents).
- **Drop the pinned `Team.Workstream` default**; add a deprecation **shim tier** in
  `resolveTeamWorkstreamName` (reads the pin + emits a notice; scheduled removal in 2.1).
- **`rm` / `archive`** — the only destructive op; confirm-gated, `--yes` to skip, `archive` moves
  to `<baseRoot>/.archive/` instead of deleting.
- **Bare `amq-squad`** → multi-session status board (graceful-degrades if `amq` is missing);
  `--help`/`-h`/`help` still print usage.
- **No global JSON `schema_version` bump** — add a new `sessions` kind only.
- **`/v2` Go module path** (semantic import versioning): module line + 67 internal imports.
- **Mission Control:** `internal/state` (pure stdlib + maildir parser, fixture-tested) +
  `internal/console` (Bubble Tea); first third-party deps confined to the console packages.
